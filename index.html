Customise checkout for Payment Links

Collect additional information, taxes, or update your branding.

When you create a payment link, you can customise the look and feel of a checkout session for your user. You can also choose what type of customer information to collect and save for later.

See After a payment link payment for post-payment customisation such as redirects and receipts.

Limit the number of times a payment link can be paid

You can limit how many times a payment link can be paid. When the link reaches the limit it deactivates and customers can’t purchase. After the limit, customers see the default deactivated message or a custom message.

  • Links with subscriptions: go to Billing → Subscriptions.
  • Other links: open the link details → Payments and analytics → Recent payments.

Dashboard

  1. Create or edit a payment link.
  2. Select Limit the number of payments and set the limit.

API

curl https://api.stripe.com/v1/payment_links \
  -u "<<YOUR_SECRET_KEY>>:" \
  -d "line_items[0][price]"="{{PRICE_ID}}" \
  -d "line_items[0][quantity]"=1 \
  -d "restrictions[completed_sessions][limit]"=1

Set a custom message for deactivated links

Show a tailored message when a link is inactive.

Dashboard

  • When you create or edit a link with a limit, select Change deactivation message.
  • When deactivating a link, the modal also lets you change the message.

API

curl https://api.stripe.com/v1/payment_links \
  -u "<<YOUR_SECRET_KEY>>:" \
  -d "line_items[0][price]"="{{PRICE_ID}}" \
  -d "line_items[0][quantity]"=1 \
  --data-urlencode inactive_message="Sorry, we are out of stock for now!"

Collect customer addresses and phone numbers

Add address and phone fields to the checkout session.

Dashboard

  1. Create or edit a payment link.
  2. Select Collect customers’ addresses and choose billing only or billing + shipping.
  3. If collecting shipping addresses, select allowed countries and optionally shipping rates.
  4. To collect phone numbers, enable Require customers to provide a phone number.

API

Use billing_address_collection, shipping_address_collection with allowed_countries, and phone_number_collection.

Collect taxes

Use Stripe Tax to calculate and collect tax automatically.

Dashboard

  1. Create or edit a payment link.
  2. Select Collect tax automatically.

API

curl https://api.stripe.com/v1/payment_links \
  -u "<<YOUR_SECRET_KEY>>:" \
  -d "line_items[0][price]"="{{PRICE_ID}}" \
  -d "line_items[0][quantity]"=1 \
  -d "automatic_tax[enabled]"=true
